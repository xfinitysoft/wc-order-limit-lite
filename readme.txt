== Description ==
Order Limit for WooCommerce lets you set minimum and maximum limits on order quantity and order value. The WooCommerce limit quantity plugin prevents customers from ordering too few or too many products, making it troublesome for the storeowner to fulfill.

[youtube hhttps://www.youtube.com/watch?v=f0waWoDAg0U]

WooCommerce Order Limit plugin is useful for storeowners who need tight control over how much can be ordered from their online store. Examples include online bakery, online dairy shops, or stores running seasonal promotions.
With customizable error messages, you can notify customers about the order limit so they can adjust the order amount accordingly. You can also disable the checkout button to prevent the customer from proceeding to the cart page.

= Set Order Limit on Products =
* Create rules for specific products and set minimum and maximum order quantities
* Apply limits on number of items or total order amount
* Apply the limit accumulatively on selected products

= Set Order Limit on Product Category =

* Create rules for specific product categories and set minimum and maximum order quantities
* Apply limits on number of items or total order amount
* Apply the limit accumulatively on selected categories


= Set Order Limits on Cart =

* Set Minimum and Maximum limits on total order amount on the cart page
* Set Minimum and Maximum quantity on cart

= Hide Checkout Button =
* Hide Checkout button when rules are not met

= Custom Message to Display When Order Limit is Reached =
* Customize messages with dynamic shortcodes to notify users of the limits
* Assist users with custom messages so they can remove errors and place the order

= Use Cases of WooCommerce Order Limit Plugin =
* A small store with limited stock and production capability – online bakery for example
* Wholesale online store in which a minimum quantity needs to be configured to make up for the small margins
* Multivendor stores in which some vendors need to specify minimum maximum order quantities (PRO version is required for compatibility with WC Vendors Pro)
* Stores running promotions that need to restrict customers from buying all the products in stock

= PREMIUM FEATURES: =

= Set Order Limits for Customers =

* Create order limit rules on user level. 
* Set maximum order limits for selected customers, user roles, or guest users
* Create product level rules for specific customers

= Enable Time Span for Limits =
* Enable time spans for all order limit rules so the limits last only for that time span
* Automatically reset limits after the specified time span
* Choose from daily, weekly, monthly, yearly or custom time span

= Compatibility with WC Vendors Pro =

* Admin can set individual limit for each vendor
* Admin can give access to vendor to change Order Limit
* Vendor has the ability to change Order Limit
